# Flutter App Template

A simple Flutter application with a home page and counter functionality.

## Features

- Home page with counter
- Material 3 design
- Cross-platform support (Android, iOS, Linux, macOS, Windows, Web)

## Getting Started

### Prerequisites

- Flutter SDK (installed at ~/development/flutter)
- Git

### Running the App

```bash
cd my_flutter_app
flutter run
```

This will launch the app on the connected device or emulator.

## Project Structure

```
lib/
  └── main.dart          # Main application entry point
```
